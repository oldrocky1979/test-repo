### 5. 迁移步骤 ⏳

1. **上海服务器（源）**
   ```bash
   # 1. 打包 workspace
   tar -czf workspace-backup.tar.gz ~/.openclaw/workspace
   
   # 2. 打包配置
   tar -czf openclaw-config.tar.gz ~/.openclaw/config
   
   # 3. 传到新加坡
   scp workspace-backup.tar.gz openclaw-config.tar.gz admin@sg-server:~/
   ```

2. **新加坡服务器（目标）**
   ```bash
   # 1. 解压
   tar -xzf workspace-backup.tar.gz -C ~/
   tar -xzf openclaw-config.tar.gz -C ~/
   
   # 2. 安装依赖
   cd ~/.openclaw/workspace
   uv pip install akshare -i https://pypi.tuna.tsinghua.edu.cn/simple
   
   # 3. 启动 Gateway
   openclaw gateway start
   ```
