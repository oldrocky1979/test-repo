# 中国公司内部审计技能框架

## 一、法规依据

### 核心法规
1. **《中华人民共和国审计法》** - 审计工作基本法律
2. **《内部审计基本准则》** - 中国内部审计协会发布
3. **《企业内部控制基本规范》** - 财政部等五部委联合发布
4. **《内部审计具体准则》** (第 1-29 号) - 操作指南

### 适用范围
- 国有企业内部审计
- 民营企业内部审计
- 上市公司内部控制审计
- 非营利组织审计

## 三、审计流程（闭环）

### 阶段 1：审计准备 (Plan)

**1.1 审计立项**
- 年度审计计划
- 临时审计任务
- 管理层交办事项

**1.2 审前调查**
```
- 了解被审计单位基本情况
- 初步评估风险领域
- 收集相关制度文件
- 确定审计重点
```

### 阶段 2：审计实施 (Do)

**2.1 进场会议**
- 宣读审计通知书
- 说明审计目的和范围
- 确定联络人员
- 安排工作条件

**2.2 审计取证**

#### 财务审计取证清单
```
□ 会计凭证抽查
□ 账簿核对
□ 报表分析
□ 银行对账单核对
□ 往来款项函证
□ 存货/固定资产盘点
□ 收入确认检查
□ 成本费用核实
□ 税金计算复核
```

#### 内控审计取证清单
```
□ 组织结构图
□ 岗位职责说明书
□ 授权审批制度
□ 业务流程图
□ 关键控制点测试
□ 不相容职务分离检查
□ 信息系统权限检查
```

#### 合规审计取证清单

**2.4 问题分析与定性**

#### 问题分类
| 类别 | 说明 | 示例 |
|------|------|------|
| A 类 - 严重 | 违反法律法规，造成重大损失 | 挪用资金、虚假报销 |
| B 类 - 重要 | 违反内控制度，存在风险 | 越权审批、流程缺失 |
| C 类 - 一般 | 操作不规范，需改进 | 单据不全、记录不及时 |

#### 问题定性依据
```
1. 违反《审计法》第 XX 条
2. 违反《企业内部控制基本规范》第 XX 条
3. 违反公司《XX 管理制度》第 XX 条
4. 不符合会计准则/财务制度规定
```

## 四、审计方法库

### 4.1 财务审计方法

| 方法 | 用途 | 操作要点 |
|------|------|----------|
| 审阅法 | 检查凭证账簿 | 关注异常分录、大额交易 |
| 核对法 | 账证/账账/账实核对 | 发现差异 |
| 盘点法 | 资产核实 | 现场清点、抽样盘点 |
| 函证法 | 往来款项确认 | 独立发函、直接回复 |
| 分析性复核 | 趋势/比率分析 | 发现异常波动 |
| 穿行测试 | 流程验证 | 追踪交易全流程 |

### 4.2 内控审计方法

| 方法 | 用途 | 操作要点 |
|------|------|----------|
| 询问法 | 了解控制环境 | 访谈关键人员 |
| 观察法 | 查看控制执行 | 现场观察操作 |
| 检查法 | 检查控制文档 | 查阅制度/记录 |
| 重新执行 | 验证控制有效性 | 独立执行控制程序 |
| 抽样测试 | 控制测试 | 统计/判断抽样 |

### 4.3 数据分析方法

```python
# 审计数据分析示例

# 1. 异常交易检测
- 大额交易筛选
- 频繁交易分析
- 关联方交易识别

# 2. 趋势分析
- 同比/环比分析
- 预算执行分析
- 比率分析

# 3. 风险预警
- 资金异常流动
- 费用异常增长
- 应收账款账龄
```

## 五、常见问题库

### 5.1 财务管理类

| 问题 | 风险 | 审计建议 |
|------|------|----------|
| 费用报销无发票 | 税务风险 | 完善报销制度，加强审核 |
| 资金支付越权审批 | 资金安全风险 | 明确授权权限，系统控制 |
| 往来款项长期挂账 | 坏账风险 | 定期清理，建立催收机制 |
| 收入确认不规范 | 财务失真 | 按会计准则确认收入 |

### 5.2 内控管理类

| 问题 | 风险 | 审计建议 |
|------|------|----------|
| 不相容职务未分离 | 舞弊风险 | 调整岗位设置 |
| 合同管理不规范 | 法律风险 | 建立合同管理制度 |
| 采购流程不透明 | 廉洁风险 | 推行阳光采购 |
| 信息系统权限混乱 | 数据安全风险 | 清理权限，定期复核 |

### 5.3 合规管理类

| 问题 | 风险 | 审计建议 |
|------|------|----------|
| 关联交易未披露 | 合规风险 | 完善披露程序 |
| 重大事项未集体决策 | 决策风险 | 执行三重一大制度 |
| 劳动用工不规范 | 法律风险 | 规范劳动合同管理 |

## 八、职业道德规范

### 审计人员守则
1. **独立性** - 保持客观公正，回避利益冲突
2. **保密性** - 保守审计秘密，不泄露信息
3. **专业性** - 持续学习，保持专业胜任能力
4. **廉洁性** - 不得收受被审计单位财物
