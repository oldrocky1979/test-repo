# A 股报告生成规则

## 📋 总则

**生效日期**: 2026-03-13  
**适用范围**: 所有 A 股股票分析报告  
**执行要求**: 必须严格执行，不得随意更改样式、章节结构、排版规范

---

## 🎯 三大标准报告类型

| 报告类型 | 使用场景 | 样式标准 | 模板文件 |
|---------|---------|---------|---------|
| 📊 **综合盘面分析** | 首次分析某股票或定期综合评估 | 宗申动力 (001696) 样式 | `templates/股票综合盘面分析_模板.html` |
| 📋 **做 T 方案** | 日内交易策略制定 | 北京君正 (300223) 样式 | `templates/A 股做 T 方案_空白模板.html` |
| 📈 **收盘复盘** | 每日盘后复盘总结 | 北京君正 (300223) 样式 | `templates/A 股收盘复盘_空白模板.html` |

## 📐 样式规范（必须严格执行）

### 一、综合盘面分析样式（宗申动力标准）

#### 1. 字体规范
```css
正文字体：14px, line-height 1.8, "Microsoft YaHei", "SimSun", Arial
标题 H1：28px 居中 + 3px #2563eb 蓝色底边
标题 H2：20px + 左侧 4px #2563eb 蓝色竖条（无背景）
标题 H3：16px #374151 灰色
```

#### 2. 表格规范
```css
表头背景：#f3f4f6 浅灰色
表头文字：#1f2937 深灰色，加粗
偶数行背景：#f9fafb 浅灰色
边框：1px #e5e7eb
```

#### 3. 颜色规范
```css
价格上涨：#dc2626 红色，加粗
价格下跌：#16a34a 绿色，加粗
中性/等待：#6b7280 灰色
```

#### 4. 框体规范
| 框体类型 | 背景色 | 边框色 | 用途 |
|---------|--------|--------|------|
| 信息框 | #f0f9ff | #bae6fd | 数据说明、提示 |
| 警告框 | #fef3c7 | #fcd34d | 风险提示 |
| 成功框 | #dcfce7 | #86efac | 利好信息 |
| 危险框 | #fee2e2 | #fca5a5 | 止损位 |
| 主要框 | #dbeafe | #93c5fd | 操作策略 |

#### 5. 特殊元素
```css
评分框：橙色渐变 (linear-gradient 135deg, #fbbf24 → #f59e0b)
评级框：橙色 #fbbf24 背景，白色文字
指标卡片：2x2 grid, #f8fafc 背景，圆角 8px
```

## 📋 章节结构（必须完整，不得删减）

### 一、综合盘面分析（8 个章节）

```
📈 一、基本统计分析
  - 最新收盘价、180 天最高/最低价、180 天涨跌幅
  - 平均成交量、当前价格在 180 天区间的位置
  - 4 个指标卡片网格展示

📊 二、趋势分析
  - MA5/MA10/MA20/MA60 均线数值
  - 股价相对均线位置（上方/下方）
  - 均线得分（0-4 分）

🔧 三、技术指标分析
  - RSI(14) 数值及信号
  - MACD 线、Signal 线、Histogram
  - 金叉/死叉状态

📅 四、近 10 个交易日走势
  - 日期、收盘价、涨跌幅、成交量
  - 近 5 日/10 日/20 日趋势统计

🎯 五、支撑位与压力位
  - 短期 (20 日) 支撑/压力
  - 中期 (60 日) 支撑/压力
  - 整数关口

📊 六、量化综合评分
  - 综合评分（0-10 分，橙色渐变框）
  - 评分明细表（均线、RSI、MACD、趋势、位置、行业属性）
  - 评级框（买入/卖出/持有观望）

💡 七、量化交易建议
  - 操作策略、建议仓位
  - 入场点位、止损位、止盈位
  - 风险提示

📋 八、数据来源
  - 数据源、股票代码、板块、行业、概念
```

---

### 二、做 T 方案（8 个章节）

```
📈 一、当前形势判断
  - 趋势、位置、振幅、成交量
  - 信息框列表展示

💰 二、关键价位
  - 阻力位 R3/R2/R1（红色标签）
  - 当前价格（蓝色标签）
  - 支撑位 S1/S2/S3（绿色标签）
  - 居中垂直排列

📊 三、做 T 策略方案
  - 方案 A：正 T（先买后卖，推荐）
    * 适用场景
    * 买入①/买入②/卖出①/卖出②（步骤表格）
    * 预期收益（成功框）
    * 止损位（警告框）
  - 方案 B：反 T（先卖后买，需底仓≥5000 股）
    * 适用条件
    * 卖出/买回（步骤表格）
    * 预期收益（成功框）

⏰ 四、分时图操作时间表
  - 09:25-15:00 各时间段策略
  - 表格形式：时间段、策略、说明

📋 五、今日操作清单
  - 时间、操作、价格、状态
  - 表格形式，高亮关键操作

⚠️ 六、风险控制
  - 跌破支撑位、快速拉升、成交量萎缩
  - 手续费、T 飞风险、T 套风险
  - 警告框列表

📝 七、情景推演
  - 低开/高开/平开三种情景
  - 概率、策略、预期收益

📊 八、复盘记录模板
  - 今日做 T 次数、成功/失败
  - 盈亏金额、改进点
  - 表格形式（项目、填写内容）
```

---

### 三、收盘复盘（5 个章节）

```
📈 一、全天走势复盘
  - 分时走势分析（时间线，8 个时间点）
  - 关键时点回顾（表格：时间、事件、价格、点评）
  - 今日做 T 机会回顾（成功框）

📊 二、技术面分析
  - K 线形态（K 线框：下影线、实体、上影线、形态解读）
  - 关键价位（阻力位 R3/R2/R1、收盘价、支撑位 S1/S2/S3）
  - 量能分析（成交量、成交额、换手率、量比、结论）

💰 三、资金面分析
  - 主力资金流向（早盘/上午/午后/尾盘/全天）
  - 流向、金额、解读（表格）
  - 解读（信息框）

📋 四、明日做 T 方案
  - 核心判断（趋势、振幅预期、做 T 空间、风险等级）
  - 关键价位预测（阻力位/昨收/支撑位）
  - 方案 A：正 T（推荐，表格 + 收益 + 止损）
  - 方案 B：反 T（表格 + 收益 + 止损）
  - 分时操作时间表（9 个时间段）
  - 风险控制（警告框列表）
  - 明日操作清单（表格）

📝 五、今日复盘总结
  - 评估表（今日建议执行度、尾盘走势、做 T 机会、明日预期、推荐策略）
  - 明日核心策略（成功框，5 点列表）
  - 一句话总结
```

## 🔄 生成流程（必须严格执行）

### 步骤 1：选择模板
根据报告类型选择对应模板：
- 综合盘面分析 → `templates/股票综合盘面分析_模板.html`
- 做 T 方案 → `templates/A 股做 T 方案_空白模板.html`
- 收盘复盘 → `templates/A 股收盘复盘_空白模板.html`

### 步骤 2：复制模板
```bash
cp ~/.openclaw/workspace/templates/[模板文件] \
   ~/.openclaw/workspace/reports/[股票名称]_[代码]_[报告类型].html
```

### 步骤 3：填充数据
- 替换 `股票名称` 为实际名称（如：盛通股份）
- 替换 `股票代码` 为实际代码（如：000823）
- 替换 `___` 为实际数据值
- 替换 `YYYY 年 MM 月 DD 日` 为实际日期
- **不得删减任何章节**
- **不得更改章节顺序**

### 步骤 4：生成 PDF
```bash
cd ~/.openclaw/workspace/reports
node generate_pdf.js [股票名称]_[代码]_[报告类型].html
```

### 步骤 5：质量检查
生成后必须检查：
- ✅ 所有章节完整
- ✅ 样式无错乱
- ✅ 数据无遗漏
- ✅ 颜色正确（涨红跌绿）
- ✅ 表格对齐

## 📊 评级标准（统一执行）

### 综合盘面分析评级

| 综合评分 | 评级 | 操作策略 | 建议仓位 |
|---------|------|---------|---------|
| 8-10 分 | 🟢 强烈买入 | 积极建仓 | 70-90% |
| 6-7 分 | 🟡 买入 | 逢低布局 | 50-70% |
| 4-5 分 | ⚪ 持有观望 | 等待方向 | 30-50% |
| 2-3 分 | 🟠 减持 | 逢高减仓 | 10-30% |
| 0-1 分 | 🔴 卖出 | 清仓离场 | 0-10% |

### 做 T 方案情景概率

| 情景 | 概率 | 策略 | 预期收益 |
|------|------|------|---------|
| 低开 (-1% 至 -2%) | 待评估 | 正 T 为主 | 待计算 |
| 高开 (+1% 至 +2%) | 待评估 | 反 T 为主 | 待计算 |
| 平开 (±0.5%) | 待评估 | 等待方向 | 待计算 |
