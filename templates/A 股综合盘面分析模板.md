# A 股综合盘面分析模板

## 模板说明

这是标准化的 A 股综合盘面分析报告模板，基于宗申动力 (001696) 样式设计。

## 使用方法

### 1. 复制模板文件
```bash
cp ~/.openclaw/workspace/templates/股票综合盘面分析_模板.html \
   ~/.openclaw/workspace/reports/股票名称_代码_综合盘面分析.html
```

### 2. 替换占位符
- `股票名称` → 实际股票名称（如：盛通股份）
- `股票代码` → 实际股票代码（如：000823）
- `YYYY 年 MM 月 DD 日` → 报告日期
- `___` → 实际数据值

### 3. 生成 PDF
```bash
cd ~/.openclaw/workspace/reports
node generate_pdf.js 股票名称_代码_综合盘面分析.html
```

## 章节结构

### 📈 一、基本统计分析
- 最新收盘价
- 180 天最高价
- 180 天最低价
- 180 天涨跌幅
- 平均成交量
- 当前价格在 180 天区间的位置

### 📊 二、趋势分析
- MA5/MA10/MA20/MA60 均线数值
- 股价相对均线位置
- 均线得分（0-4 分）

### 🔧 三、技术指标分析
- RSI(14) 数值及信号
- MACD 线、Signal 线、Histogram
- 金叉/死叉状态

### 📅 四、近 10 个交易日走势
- 日期、收盘价、涨跌幅、成交量
- 近 5 日/10 日/20 日趋势统计

### 🎯 五、支撑位与压力位
- 短期 (20 日) 支撑/压力
- 中期 (60 日) 支撑/压力
- 整数关口

### 📊 六、量化综合评分
- 综合评分（0-10 分）
- 评分明细：
  - 均线系统
  - RSI
  - MACD
  - 趋势
  - 位置
  - 行业属性

### 💡 七、量化交易建议
- 操作策略（买入/卖出/持有观望）
- 建议仓位
- 入场点位
- 止损位
- 止盈位
- 风险提示

### 📋 八、数据来源
- 数据源：AkShare (东方财富)
- 股票代码及名称
- 板块信息
- 行业分类
- 概念标签

## 评级标准

| 综合评分 | 评级 | 操作策略 |
|---------|------|---------|
| 8-10 分 | 🟢 强烈买入 | 70-90% 仓位 |
| 6-7 分 | 🟡 买入 | 50-70% 仓位 |
