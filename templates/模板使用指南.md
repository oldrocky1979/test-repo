## 📊 综合盘面分析（8 章节）

### 使用场景
- 首次分析某只股票
- 定期综合评估（周/月）
- 量化交易建议

### 章节结构（不得删减）

```
📈 一、基本统计分析
  ├─ 最新收盘价
  ├─ 180 天最高/最低价
  ├─ 180 天涨跌幅
  ├─ 平均成交量
  └─ 4 个指标卡片网格展示

📊 二、趋势分析
  ├─ MA5/MA10/MA20/MA60 均线数值
  ├─ 股价相对均线位置
  └─ 均线得分（0-4 分）

🔧 三、技术指标分析
  ├─ RSI(14) 数值及信号
  ├─ MACD 线、Signal 线、Histogram
  └─ 金叉/死叉状态

📅 四、近 10 个交易日走势
  ├─ 日期、收盘价、涨跌幅、成交量
  └─ 近 5 日/10 日/20 日趋势统计

🎯 五、支撑位与压力位
  ├─ 短期 (20 日) 支撑/压力
  ├─ 中期 (60 日) 支撑/压力
  └─ 整数关口

📊 六、量化综合评分
  ├─ 综合评分（0-10 分，橙色渐变框）
  ├─ 评分明细表
  └─ 评级框（买入/卖出/持有观望）

💡 七、量化交易建议
  ├─ 操作策略、建议仓位
  ├─ 入场点位、止损位、止盈位
  └─ 风险提示

📋 八、数据来源
  ├─ 数据源
  ├─ 股票代码、板块
  └─ 行业、概念
```

### 样式规范（宗申动力标准）

**字体**:
- 正文：14px, line-height 1.8, "Microsoft YaHei"
- H1: 28px 居中 + 3px #2563eb 蓝色底边
- H2: 20px + 左侧 4px #2563eb 蓝色竖条
- H3: 16px #374151 灰色

**颜色**:
- 价格上涨：#dc2626 红色，加粗
- 价格下跌：#16a34a 绿色，加粗

**特殊元素**:
- 评分框：橙色渐变 (linear-gradient 135deg, #fbbf24 → #f59e0b)
- 指标卡片：2x2 grid, #f8fafc 背景，圆角 8px

## 📋 做 T 方案（8 章节）

### 使用场景
- 日内交易策略制定
- 每日盘前推送
- 做 T 操作指导

### 章节结构（不得删减）

```
📈 一、当前形势判断
  ├─ 趋势判断
  ├─ 位置判断
  ├─ 振幅评估
  └─ 成交量分析

💰 二、关键价位
  ├─ 阻力位 R3/R2/R1（红色标签）
  ├─ 当前价格（蓝色标签）
  └─ 支撑位 S1/S2/S3（绿色标签）
  └─ 居中垂直排列

📊 三、做 T 策略方案
  ├─ 方案 A：正 T（先买后卖，推荐）
  │   ├─ 适用场景
  │   ├─ 买入①/买入②/卖出①/卖出②（步骤表格）
  │   ├─ 预期收益（成功框）
  │   └─ 止损位（警告框）
  └─ 方案 B：反 T（先卖后买，需底仓≥5000 股）
      ├─ 适用条件
      ├─ 卖出/买回（步骤表格）
      └─ 预期收益（成功框）

⏰ 四、分时图操作时间表
  ├─ 09:25-09:35 观察集合竞价
  ├─ 09:35-10:30 第一操作窗口
  ├─ 10:30-11:30 第二操作窗口
  ├─ 11:30-13:00 午休
  ├─ 13:00-13:30 观察开盘方向
  ├─ 13:30-14:30 第三操作窗口（最佳）
  ├─ 14:00-14:30 谨慎观望
  └─ 14:30-15:00 平仓时间

📋 五、今日操作清单
  └─ 时间、操作、价格、状态（表格）

⚠️ 六、风险控制
  ├─ 跌破支撑位、快速拉升、成交量萎缩
  ├─ 手续费、T 飞风险、T 套风险
  └─ 警告框列表

📝 七、情景推演
  ├─ 低开情景（概率、策略、预期收益）
  ├─ 高开情景（概率、策略、预期收益）
  └─ 平开情景（概率、策略、预期收益）

📊 八、复盘记录模板
  ├─ 今日做 T 次数、成功/失败
  ├─ 盈亏金额、改进点
  └─ 表格形式
```

### 样式规范（北京君正标准）

**字体**:
- 正文：12px, line-height 1.6
- H1: 22px #1a5276 深蓝 + 2px 底边
- H2: 16px + 浅蓝背景 #ebf5fb + 左侧 4px 蓝条
- H3: 14px #1a5276 深蓝

**价格标签**:
- 阻力位：#ffebee 背景，#c62828 红色文字
- 支撑位：#e8f5e9 背景，#2e7d32 绿色文字
- 当前价：#1a5276 蓝色背景，白色文字

**策略框**:
- 正 T 方案：#f0f8ff 浅蓝背景，2px #2874a6 蓝色边框
- 反 T 方案：#fff8f0 浅橙背景，2px #e67e22 橙色边框

## 📈 收盘复盘（5 章节）

### 使用场景
- 每日盘后复盘总结
- 15:40 后生成
- 次日做 T 方案参考

### 章节结构（不得删减）

```
📈 一、全天走势复盘
  ├─ 分时走势分析（时间线，8 个时间点）
  ├─ 关键时点回顾（表格：时间、事件、价格、点评）
  └─ 今日做 T 机会回顾（成功框）

📊 二、技术面分析
  ├─ K 线形态（K 线框：下影线、实体、上影线、形态解读）
  ├─ 关键价位（阻力位 R3/R2/R1、收盘价、支撑位 S1/S2/S3）
  └─ 量能分析（成交量、成交额、换手率、量比、结论）

💰 三、资金面分析
  ├─ 主力资金流向（早盘/上午/午后/尾盘/全天）
  ├─ 流向、金额、解读（表格）
  └─ 解读（信息框）

📋 四、明日做 T 方案
  ├─ 核心判断（趋势、振幅预期、做 T 空间、风险等级）
  ├─ 关键价位预测（阻力位/昨收/支撑位）
  ├─ 方案 A：正 T（推荐，表格 + 收益 + 止损）
  ├─ 方案 B：反 T（表格 + 收益 + 止损）
  ├─ 分时操作时间表（9 个时间段）
  ├─ 风险控制（警告框列表）
  └─ 明日操作清单（表格）

📝 五、今日复盘总结
  ├─ 评估表（今日建议执行度、尾盘走势、做 T 机会、明日预期、推荐策略）
  ├─ 明日核心策略（成功框，5 点列表）
  └─ 一句话总结
```

### 样式规范

**与做 T 方案完全一致**（北京君正标准）

**特殊元素**:
- 时间线：左侧 3px #2874a6 蓝色竖线，圆点标记 (●)
- K 线框：#f8f9fa 灰色背景，圆角 6px，居中

## 🔄 标准生成流程

### 步骤 1：选择模板
根据报告类型选择对应模板文件

### 步骤 2：复制模板
```bash
cp /home/admin/.openclaw/workspace/github-sync/templates/[模板文件] \
   /home/admin/.openclaw/workspace/reports/[股票名称]_[代码]_[报告类型].html
```

### 步骤 3：填充数据
- 替换 `股票名称` 为实际名称
- 替换 `股票代码` 为实际代码
- 替换 `___` 为实际数据值
- 替换 `YYYY 年 MM 月 DD 日` 为实际日期
- ⚠️ **不得删减任何章节**
- ⚠️ **不得更改章节顺序**

### 步骤 4：生成 PDF
```bash
cd /home/admin/.openclaw/workspace/skills/chinese-document-generator

# 使用 html-pdf
node -e "const pdf = require('html-pdf'); pdf.create(htmlContent, {format: 'A4'}).toFile('output.pdf', callback);"
```

### 步骤 5：质量检查
- ✅ 所有章节完整（100%）
- ✅ 样式无错乱
- ✅ 数据无遗漏
- ✅ 颜色正确（涨红跌绿）
- ✅ 表格对齐
