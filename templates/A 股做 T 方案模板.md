# A 股做 T 方案模板

## 模板说明

这是标准化的 A 股做 T 交易方案模板，基于北京君正 (300223) 样式设计。

## 使用方法

### 1. 复制模板文件
```bash
cp ~/.openclaw/workspace/templates/A 股做 T 方案_空白模板.html \
   ~/.openclaw/workspace/reports/股票名称_代码_做 T 方案.html
```

### 2. 替换占位符
- `股票名称` → 实际股票名称（如：盛通股份）
- `股票代码` → 实际股票代码（如：000823）
- `___` → 实际数据值
- `YYYY 年 MM 月 DD 日 HH:MM` → 实际生成时间

### 3. 生成 PDF
```bash
cd ~/.openclaw/workspace/reports
node generate_pdf.js 股票名称_代码_做 T 方案.html
```

## 章节结构

### 📈 一、当前形势判断
- 趋势判断
- 位置判断
- 振幅评估
- 成交量分析

### 💰 二、关键价位
- 阻力位 R3/R2/R1
- 当前价格
- 支撑位 S1/S2/S3

### 📊 三、做 T 策略方案
- **方案 A：正 T（推荐）** - 先买后卖
  - 买入①/买入②
  - 卖出①/卖出②
  - 预期收益
  - 止损位
- **方案 B：反 T** - 先卖后买（需底仓≥5000 股）
  - 卖出
  - 买回
  - 预期收益

### ⏰ 四、分时图操作时间表
- 09:25-09:35 观察集合竞价
- 09:35-10:30 第一操作窗口
- 10:30-11:30 第二操作窗口
- 11:30-13:00 午休
- 13:00-13:30 观察开盘方向
- 13:30-14:30 第三操作窗口（最佳）
- 14:00-14:30 谨慎观望
- 14:30-15:00 平仓时间

### 📋 五、今日操作清单
- 时间、操作、价格、状态

### ⚠️ 六、风险控制
- 跌破支撑位处理
- 快速拉升处理
- 成交量萎缩处理
- 手续费考虑
- T 飞风险
- T 套风险

### 📝 七、情景推演
- 低开情景
- 高开情景
- 平开情景

### 📊 八、复盘记录模板
- 做 T 次数
- 成功/失败
- 盈亏金额
- 改进点

## 操作建议

| 情景 | 策略 | 仓位 |
|------|------|------|
| 低开 (-1% 至 -2%) | 正 T 为主 | 50-70% |
| 高开 (+1% 至 +2%) | 反 T 为主 | 30-50% |
| 平开 (±0.5%) | 等待方向 | 20-30% |
