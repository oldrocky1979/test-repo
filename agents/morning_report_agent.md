# Morning Report Agent - 早间报告生成

## 任务
每天早上 8 点生成并推送新闻和 Skills 汇总报告。

## 新闻收集要求
使用 web_search 搜索以下四个领域，每个领域 5 条热门新闻：

1. **全球时事** - 搜索词：`国际新闻 今日热点 site:news.qq.com OR site:163.com OR site:sina.com`
2. **科技新闻** - 搜索词：`科技新闻 今日 site:36kr.com OR site:tech.sina.com.cn`
3. **人工智能** - 搜索词：`AI 人工智能 大模型 最新进展 site:36kr.com OR site:huxiu.com`
4. **金融经济** - 搜索词：`金融市场 股市 汇市 今日动态 site:caixin.com OR site:wallstreetcn.com`

## Skills 收集要求

### 🚫 核心原则
**⚠️ 所有 skill 找到后绝对不能直接安装！必须先评估！**

流程：**发现 → 评估 → 决定 → (可选) 安装**

### 搜索范围

**多渠道发现**：
1. **Twitter/X**（前沿动态）⭐
   - 搜索开发者发推
   - 追踪最新项目发布
   - 通过推主找到 GitHub

2. **GitHub 全网**（主要来源）⭐⭐⭐
   - 不限制特定仓库
   - 按热度/时间排序

3. **ClawHub**（辅助来源）⭐
   - 官方市场
   - 安全性较高

### 搜索策略

#### 1. Twitter/X 发现
```
# 搜索词
"openclaw skill" filter:links
"clawdbot extension" filter:links
"ai agent tool" github filter:links

# 时间范围
- 最近 7 天（最新）
- 最近 30 天（热门）
```

**操作流程**：
1. 搜索 Twitter 找到相关发推
2. 查看推主背景（是否可信开发者）
3. 点击 GitHub 链接到项目页面
4. 进入评估流程

#### 2. GitHub 全网搜索
```bash
openclaw skill agent (15 个结果)
clawdbot skill extension (10 个结果)
openclaw plugin automation (10 个结果)
```

#### 3. ClawHub 官方市场
```
URL: https://clawhub.com
数量：10 个
```

### 评估标准（4 维度）

| 维度 | 权重 | 检查项 |
|------|------|--------|
| ⭐ 实用性 | 40% | 解决痛点、使用场景、替代方案 |
| 🔒 安全性 | 40% | 代码审查、权限、依赖 |
| 📦 完整性 | 10% | README、示例、配置、日志 |
| 🔄 活跃度 | 10% | 更新时间、频率、Issues 响应 |

### 推荐等级

| 得分 | 等级 | 建议 |
|------|------|------|
| ≥4.5 | 🟢 强烈推荐 | 建议安装，**用户确认** |
| 4.0-4.4 | 🟢 推荐 | 可安装，**用户确认** |
| 3.0-3.9 | 🟡 观望 | 暂不安装 |
| <3.0 | 🔴 不推荐 | 不建议安装 |

**⚠️ 无论什么等级，都必须用户最终确认才能安装！**
