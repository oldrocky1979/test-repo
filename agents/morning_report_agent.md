# Morning Report Agent - 早间报告生成

## 任务
每天早上 8 点生成并推送新闻和 Skills 汇总报告。

## 新闻收集要求
使用 web_search 搜索以下四个领域，每个领域 5 条热门新闻：

1. **全球时事** - 搜索词：`国际新闻 今日热点 site:news.qq.com OR site:163.com OR site:sina.com`
2. **科技新闻** - 搜索词：`科技新闻 今日 site:36kr.com OR site:tech.sina.com.cn`
3. **人工智能** - 搜索词：`AI 人工智能 大模型 最新进展 site:36kr.com OR site:huxiu.com`
4. **金融经济** - 搜索词：`金融市场 股市 汇市 今日动态 site:caixin.com OR site:wallstreetcn.com`

## Skills 收集要求
1. 使用 `agents_list` 获取可用技能列表
2. 搜索 ClawHub 热门技能
3. 评估每个技能的实用性和安全性
