## 2.2 核心法规解读

### 《审计署关于内部审计工作的规定》（第 11 号令）

#### 关键条款
| 条款 | 内容要点 | 实务影响 |
|------|----------|----------|
| 第 3 条 | 内部审计定义 | 确认 + 咨询双重职能 |
| 第 6 条 | 独立性要求 | 直接向党组织/董事会报告 |
| 第 12 条 | 审计权限 | 查阅资料、参加 meetings、调查权 |

| 第 20 条 | 结果运用 | 整改要求、考核挂钩、追责依据 |
| 第 23 条 | 监督管理 | 审计机关对内部审计指导和监督 |

## 2.4 职业道德规范

### 四大原则
```
┌──────────────────────────────────────────────────────┐
│               内部审计职业道德                        │
├────────────┬────────────┬────────────┬───────────────┤
│   诚信     │   客观     │   保密     │   胜任能力    │
├────────────┼────────────┼────────────┼───────────────┤
│ 诚实守信   │ 公正判断   │ 保护信息   │ 持续学习      │
│ 不弄虚作假 │ 不受影响   │ 不泄露    │ 保持资质      │
│ 言行一致   │ 回避利益冲突│ 合规使用  │ 遵守准则      │
└────────────┴────────────┴────────────┴───────────────┘
```

### 禁止行为清单
- [ ] 参与可能影响独立性的活动
- [ ] 接受被审计单位的礼品、招待（超过象征性价值）
- [ ] 泄露审计过程中获知的机密信息
- [ ] 利用职务之便谋取私利
- [ ] 出具虚假或不实的审计结论
- [ ] 超越权限做出承诺或决定

### 利益冲突申报
审计人员需申报以下情况：
1. 在被审计单位持有股份或其他经济利益
2. 近亲属在被审计单位任职
3. 曾在被审计单位任职（2 年内）
4. 与被审计单位存在其他可能影响独立性的关系

## 2.5 审计权限与责任

### 法定权限
| 权限类型 | 具体内容 | 限制条件 |
|----------|----------|----------|
| 资料查阅权 | 查阅所有相关文件、记录 | 保密义务 |
| 参加会议权 | 参加与审计事项有关的会议 | 不干预决策 |
| 调查取证权 | 向有关单位和个人调查 | 合法合规 |
| 封存权 | 封存可能被转移的资料 | 需审批 |
| 建议权 | 提出改进建议 | 不直接执行 |
| 报告权 | 直接向最高层报告 | 重大紧急事项 |

### 审计责任
- **审计责任**：对审计结论的真实性负责
- **被审计单位责任**：对提供资料的真实性、完整性负责
- **责任划分**：审计不能替代管理责任
