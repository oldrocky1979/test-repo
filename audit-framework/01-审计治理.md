# 第一章：审计治理与组织架构

## 1.1 内部审计定位

### 定义
内部审计是一种独立、客观的确认和咨询活动，通过系统化、规范化的方法，评价和改进风险管理、控制和治理过程的效果，帮助组织实现其目标。

### 三大职能
```
┌─────────────────────────────────────────────────────────┐
│                    内部审计职能                         │
├─────────────┬─────────────┬─────────────────────────────┤
│   确认服务   │   咨询服务   │       评价服务             │
├─────────────┼─────────────┼─────────────────────────────┤
│ • 财务审计   │ • 流程优化   │ • 内部控制评价             │
│ • 合规审计   │ • 系统设计   │ • 风险管理评估             │
│ • 运营审计   │ • 培训指导   │ • 治理效果评价             │
│ • 舞弊调查   │ • 最佳实践   │ • 绩效评价                 │
└─────────────┴─────────────┴─────────────────────────────┘
```

### 汇报关系原则
| 汇报类型 | 汇报对象 | 频率 | 内容 |
|----------|----------|------|------|
| 职能汇报 | 审计委员会 | 季度 | 审计计划、重大发现、资源需求 |
| 行政汇报 | CEO/CFO | 月度 | 日常运营、人员管理 |
| 紧急汇报 | 审计委员会主席 | 即时 | 重大舞弊、严重违规 |

## 1.3 独立性保障

### 组织独立性
- [ ] 内部审计部门独立于被审计单位
- [ ] 首席审计官直接向审计委员会报告
- [ ] 审计预算单独列支，不受被审计单位影响
- [ ] 审计人员薪酬由审计委员会审定

### 个人独立性
- [ ] 审计人员不得审计自己曾经负责的业务（至少 2 年冷却期）
- [ ] 审计人员与被审计单位无利益关系
- [ ] 定期声明独立性（年度签署独立性声明）
- [ ] 审计项目实行轮换制（项目负责人 3-5 年轮换）

### 独立性威胁及应对
| 威胁类型 | 表现 | 应对措施 |
|----------|------|----------|
| 自我复核 | 审计自己设计/执行的系统 | 回避、第三方复核 |
| 利益冲突 | 与被审计单位有经济利益 | 披露、回避 |
| 熟悉威胁 | 与被审计单位关系过密 | 轮换、复核 |
| 胁迫威胁 | 受到压力影响判断 | 升级报告、保护机制 |

## 1.4 审计委员会职责

### 核心职责
1. **监督财务报告**：审核财务报表、会计政策变更
2. **监督内部控制**：评价内控有效性、审阅内控报告
3. **监督内部审计**：审批审计计划、听取审计汇报
4. **监督外部审计**：选聘外部审计师、协调内外审计
5. **监督合规管理**：审阅合规报告、重大违规事项

### 会议机制
- **频率**：至少每季度召开一次
- **法定人数**：过半数委员出席
- **列席人员**：首席审计官、财务总监、外部审计师
- **单独会议**：每年至少一次与内部审计、外部审计单独会议

## 1.5 内部审计章程

### 必备内容
```markdown
1. 部门使命与目标
2. 组织定位与汇报关系
3. 审计范围与权限
4. 独立性保障措施
5. 职业道德规范
6. 审计标准与方法
7. 质量保证机制
8. 章程修订程序
```

### 审批流程
1. 首席审计官起草
2. 审计委员会审议
3. 董事会批准
4. 定期复审（至少每 3 年一次）

## 1.6 资源配置

### 人员配置参考
| 企业规模 | 审计人员配置 | 专业结构建议 |
|----------|--------------|--------------|
| 小型 (<10 亿) | 2-5 人 | 财务 + 运营 |
| 中型 (10-100 亿) | 5-15 人 | 财务 + 运营+IT+ 合规 |
| 大型 (>100 亿) | 15-50 人 | 专业化分组 + 专家支持 |

### 能力框架
- **核心能力**：会计、审计、财务分析
- **业务能力**：行业知识、业务流程、风险管理
- **技术能力**：数据分析、IT 审计、信息系统
- **软技能**：沟通、谈判、项目管理、写作

## 1.7 关键绩效指标 (KPI)

### 部门级 KPI
| 指标 | 计算公式 | 目标值 |
|------|----------|--------|
| 审计计划完成率 | 实际完成项目/计划项目×100% | ≥95% |
| 审计建议采纳率 | 被采纳建议/总建议×100% | ≥85% |
| 整改完成率 | 已完成整改/应整改项×100% | ≥90% |
| 客户满意度 | 被审计单位评分平均值 | ≥4.0/5.0 |
| 审计覆盖率 | 已审计单位/应审计单位×100% | 按风险确定 |

### 项目级 KPI
- 项目按时完成率
- 预算执行率
- 重大发现数量
- 报告质量评分
